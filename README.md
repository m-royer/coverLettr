# CoverLetter

Small app for writing cover letters quickly.

[https://coverlettr.com](https://coverlettr.com/)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
